Tangzon 产品管理工具 v3.5.15 升级说明

1. 修复更新报错提示
- 当 GitHub / 更新服务器暂时连不上时，软件现在会显示更清楚的网络提示。
- 这类报错不会影响产品库、待分配区、活动产品或本地数据。
- 原始错误仍会保留在提示详情里，方便后续排查网络问题。

2. 小更新备用线路
- 在线小更新会先连接 GitHub 原始地址。
- 如果 GitHub 原始地址超时，会自动尝试备用 CDN 地址。
- 小更新文件仍然需要通过 SHA-256 校验，备用线路不会降低安全性。

3. 使用建议
- 如果看到 net::ERR_CONNECTION_TIMED_OUT、SSL/TLS handshake failed，大多是当前网络到 GitHub 不稳定。
- 可以稍后重试，或检查代理/VPN/公司网络限制。
- GitHub Desktop 推送失败属于同一个网络层问题，不是 Tangzon 数据或软件功能损坏。

4. 说明文件规则
- 本文件继续按你的要求固定覆盖，不再新增“升级说明_vX.X.X”文件。
