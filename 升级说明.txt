Tangzon 产品管理工具 v3.5.9 升级说明

1. 抓取解析增强
- 新增 Cheerio 可选解析器：星级、评论数、价格、主图会优先使用更稳定的 HTML 结构解析。
- 如果依赖没有安装成功，会自动回退到原来的页面解析方式，不影响软件打开和使用。
- 仍保持安全抓取策略：只直连 Amazon UK，不使用代理池或规避风控方案，遇到异常自动退避。

2. 官方接口预留
- 设置页新增 Amazon SP-API 官方接口入口。
- 当前只保存“是否启用”和 Marketplace ID，不保存密钥，避免误存敏感信息。
- 后续可在这个基础上继续接正式授权和官方数据同步。

3. 本地缓存与性能地基
- 新增 Dexie 可选本地缓存基础层，为后续更大的操作日志、备份索引、性能优化做准备。
- Dexie 没安装时会自动跳过，不影响现有 IndexedDB 图片存储。

4. 自动测试
- 新增 Playwright 冒烟测试，用于自动检查页面能否打开、产品区和设置入口是否存在。
- 新增 npm 脚本：npm run test:smoke。

5. 注意
- 本次已更新 package.json 依赖项，但当前环境下载依赖时网络连接中断。
- 网络恢复后请在软件目录执行 npm install，再运行 npm run test:smoke 验证自动测试。
