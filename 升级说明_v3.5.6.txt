Tangzon 产品管理工具 v3.5.6 升级说明

1. 星级抓取记录
- 抓取历史现在显示产品图片。
- ASIN 可点击跳转到 Amazon UK 商品页。
- 完整 JSON 备份包含星级抓取记录。

2. 导入 / 导出
- 导入取消“大类 / 小类必填”，留空会自动进入“未分类 / 其他”。
- 导入模板和 Excel/CSV 导出字段已同步更新。
- 支持细分类型、图片、竞品 ASIN、销量、等级、备注等字段。

3. 活动池
- 活动池顶部新增 ASIN / SKU / 新货号筛选。
- 可快速定位某个产品在哪个活动里。

4. JSON 类目导入导出
- 导出菜单新增“按类目导出 JSON”。
- 导入 JSON 时可选择完整覆盖，或只导入勾选的大类。

5. 产品卡修复
- 修复编辑弹窗修改星级 / 评论数后，产品卡显示不刷新的问题。

6. 淘汰 / 产品库 / 待分配区互通
- 产品库已有的 ASIN 会自动从淘汰列表移出。
- 待分配区会自动清理已存在于产品库的 ASIN。
- 分配待分配产品时会自动去重，避免两边重复。

7. 待分配区优化
- 待分配区导入补齐细分类型、图片、竞品、销量等字段。
- 打开待分配区时会自动自检重复数据。
