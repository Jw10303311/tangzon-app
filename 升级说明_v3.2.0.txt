 Tangzon v3.2.0 — 8 项更新
═══════════════════════════════════════════════════════

【本次更新内容】

 1. 新货号也能搜索到（之前只能搜老货号）
 2. 勾选产品后可一键批量设置细分类型
    （选中产品 → 底部蓝条 → "设置细分类型"）
 3. 修复：编辑产品卡数据后细分类型丢失的问题
 4. 有主图时也能二次手动抓图（会弹确认框防误覆盖）
    注：批量抓图仍只处理缺图产品
 5. 勾选多个产品可一起拖动排序（变体调位用）
 6. 状态栏不再遮挡产品卡；撤销栏 20 秒后自动消失
 7. 后台自动更新星级/评论数（实验性，默认关闭）
    - 在 设置 里打开"自动更新星级/评论数"
    - 每天后台慢抓约 50 个，约 2 周轮完一遍
    - 状态栏右侧会显示抓取进度
 8. 修复"检查更新"点击无反应的问题

【这是你第一次走 GitHub 自动更新流程！】

  步骤 A — 替换文件（4 个）
    把这 4 个文件复制到 C:\Users\jay\Desktop\tangzon-app\
    覆盖原文件：
      main.js
      preload.js
      package.json
      Tangzon_产品管理_个人版本.html

  步骤 B — GitHub Desktop 提交
    1. 打开 GitHub Desktop
    2. 左侧会显示 4 个文件有改动
    3. Summary 填：v3.2.0 update
    4. 点 "Commit to main"
    5. 点 "Push origin"

  步骤 C — 等 GitHub 自动打包（5-10 分钟）
    package.json 版本号变成 3.2.0 了，
    推送后 GitHub Actions 会自动打包并发布 Release。
    （也可以去 Actions 页面看进度）

  步骤 D — 你电脑上的 Tangzon 自动提示更新
    打包完成后，打开你已安装的 Tangzon
    8 秒后会自动弹窗："发现新版本 v3.2.0"
    点"立即下载" → 下完点"立即重启安装"
    完成升级！

  （以后每次升级都是这个流程，30 秒提交，剩下自动）
