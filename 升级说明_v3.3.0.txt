【本次更新】
 · 界面整体改为微信极简风 + 飞书蓝 (#3370ff) 主色
 · 去掉渐变和重阴影 → 扁平卡片 + 极淡发丝边框
 · 等级 ABCD 改用左侧细色条标识，不再用大面积底色
 · 留白加大，看着更清爽不累眼
 · 深色模式同步调整为微信暗色风
 · 修复：标题栏版本号不自动更新（以后自动跟随）

【升级步骤】
 1. 解压，5 个文件复制到 tangzon-app 文件夹覆盖：
      main.js / preload.js / package.json
      Tangzon_产品管理_个人版本.html
      .github\workflows\build.yml
 2. GitHub Desktop → Commit (填 v3.3.0) → Push
 3. 等 5-10 分钟自动打包
 4. Tangzon 自动弹更新提示 → 下载 → 重启安装
