 Tangzon v3.3.1 — 修复自动更新 404
═══════════════════════════════════════════════

【问题根因】
 自动更新能"检测"到新版本，但"下载"时报 404。
 原因：打包出的 exe 文件名里有中文「产品管理」，
 GitHub 下载链接的中文编码导致 electron-updater
 找不到文件。

【修复】
 把打包文件名改成纯英文：Tangzon-3.3.1-x64.exe
 （应用名、桌面图标、标题栏仍是中文，用户无感）

【⚠️ 重要 — 这次必须手动装一次】
 因为你现在装的版本文件名是中文的，它的自动更新
 永远会 404。必须手动装一次 v3.3.1（纯英文文件名），
 之后的 v3.3.2、v3.3.3... 自动更新就全好了。

【升级步骤】
 1. 解压，5 个文件复制到 tangzon-app 覆盖：
      main.js / preload.js / package.json
      Tangzon_产品管理_个人版本.html
      .github\workflows\build.yml
 2. GitHub Desktop → Commit (v3.3.1) → Push
 3. 等 5-10 分钟打包完成
 4. 去 Releases → v3.3.1 → 手动下载 Tangzon-3.3.1-x64.exe
 5. 双击安装（这是最后一次手动安装）
 6. 之后所有更新都能自动下载了
